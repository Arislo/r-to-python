# Py-R - RとPythonを相互に理解するための学習プラットフォーム (Astro Starlight)

Py-R は、R と Python のコードを読み比べながら理解できる  
相互学習プラットフォームとして作成したサイトです。

両言語を並べて比較しながら学べるサイトは多くありませんでした。

そこで、大学のゼミで実際に使用した分析内容をベースに、  
自分自身の備忘録としても活用できるよう、  
R と Python の対応関係を体系的にまとめたのがこの Py-R です。

来年以降の学習やデータ分析の実践でも活かせるよう、  
長期的に成長できるドキュメントとして整備しています。

---

## 🔗 URL
https://r-to-python.pages.dev/

---

## 🎯 Purpose

- R と Python を併用する現場で迷わないための「橋渡し」資料を作る  
- 両言語のコードを並べて比較し、思考の違いを理解しやすくする  
- ゼミで扱ったデータ分析内容を整理し、長期的に残す  
- 自分自身の備忘録として、学びを積み重ねられるプラットフォームにする  
- Astro Starlight を使ったドキュメントサイト構築の実践

---

## 🧩 Features

- 文法・データ構造の違いを並べて理解できる構成
- データ分析の流れをステップ形式で整理
- Markdown/MDX ベースで記事を簡単に追加・拡張可能
- Starlight のサイドバーによる読みやすいナビゲーション
- シンタックスハイライト付きコードブロック
- レスポンシブ対応
