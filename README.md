## Py-Rについて

概要
---
RとPythonのコードを相互に理解するためのプラットフォームとして作成しました。
両言語を読み比べながら分析を行うサイトがあまりなかったため、自分自身にも備忘録として作成したものです。
大学で使用したものをまとめたものになっています。来年にもこれを生かしてさらにデータ分析を頑張っていきたいと思います。

仕様技術
---
フロントエンド:Astro Starlight

フォーマッター: Prettier 

CSSフレームワーク: Tailwind CSS 

デプロイ: Cloudflare
